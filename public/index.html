Workout A: Hip Stability & Single-Leg Ground Force

1. DB Reverse Lunges

3 Sets x 10/leg. Load front heel dynamically. Builds specific push power.

2. Banded Monster Walks

3 Sets x 40s. Band around shins. Step wide laterally to strengthen hips.

3. DB Goblet Squats

3 Sets x 12. Heavy DB at chest. Focus on deep glute and core control.

4. Banded Clamshells

3 Sets x 15/side. Strengthens Gluteus Medius to keep knee track aligned.

Workout B: Posterior Propulsion & Rotational Core

1. DB Romanian Deadlifts

3 Sets x 12. Keep spine neutral. Focuses heavily on hamstrings & glutes.

2. Banded Pallof Press

3 Sets x 12/side. Resisting rotation builds powerful torso stiffness.

3. Single-Leg DB RDL

3 Sets x 8/leg. Imparts great single-leg balance and hamstring depth.

4. Banded Glute Bridges

3 Sets x 15. Band above knees. Drive hips up, pushing knees apart.

Workout C: Stride Elasticity & Ankle/Foot Rigidity

1. Single-Leg Calf Raises

3 Sets x 15. Hold DB on active side. Toughens Achilles for high bounce.

2. Banded Toe Hip-Pull

3 Sets x 12. Band around toes. Raise knee up to build hip-flexor snap.

3. DB Bulgarian Split Squats

3 Sets x 8/leg. Quad power and deep knee flex reinforcement.

4. Banded Pull-Aparts

3 Sets x 15. Strengthens posterior shoulders for deep arm drive.

Training Subphases

Complete each sequence step-by-step

Phase 1: Speed Induction (Subphases 1 - 10)

Neuromuscular conditioning. We alternate short, explosive 400m efforts with targeted glute/heel stability to safely establish a fast turnover.

Phase 2: Speed Endurance (Subphases 11 - 20)

Aerobic threshold development. We extend speed efforts to 1000m blocks and integrate posterior muscle loading (deadlifts) to safeguard joints.

Phase 3: Peak & Taper (Subphases 21 - 27)

Maximum capacity and recovery. We test with 1-mile repetitions before backing off volume so your muscles supercompensate for Thanksgiving.
